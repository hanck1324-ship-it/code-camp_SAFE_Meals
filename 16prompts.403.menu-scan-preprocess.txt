핵심요구사항) playwright 테스트를 활용하여 TDD 기반으로 구현하고, 테스트에 통과할 때까지 반복할 것.

1) 테스트 제외 라이브러리
   - jest
   - @testing-library/react

2) 테스트 조건
   - timeout은 설정하지 않거나, 500ms 미만으로 설정할 것.
   - 페이지가 완전히 로드된 후 테스트할 것.
       - 페이지 로드 식별 요구사항: 고정식별자 data-testid 대기 방법
       - **중요금지사항** 페이지 로드 식별 금지사항: networkidle 대기 방법

핵심요구사항) Canvas API를 활용한 이미지 전처리 커스텀 훅을 구현할 것.

핵심요구사항) 다음 전처리 기능을 step-by-step으로 구현하고, 적용 결과를 체크리스트로 반환할 것.
                1) 이미지 회전 보정
                    - EXIF 데이터 기반 자동 회전
                    - 수동 회전 기능 (90도 단위)

                2) 이미지 크롭
                    - 자동 경계 감지 (메뉴판 영역 인식)
                    - 수동 크롭 영역 선택 기능

                3) 대비 보정
                    - 명암 자동 조정
                    - 선명도 향상
                    - OCR 인식률 향상을 위한 최적화

                4) 결과 반환
                    - 전처리된 이미지 Blob 반환
                    - Base64 인코딩 옵션
                    - 원본 이미지 보존

핵심요구사항) 훅 인터페이스를 다음과 같이 구현할 것.
                - 입력: File | Blob
                - 출력: { preprocessedImage: Blob, base64: string, isProcessing: boolean, error: string | null }
                - 메서드: processImage(), rotate(), crop(), adjustContrast()
