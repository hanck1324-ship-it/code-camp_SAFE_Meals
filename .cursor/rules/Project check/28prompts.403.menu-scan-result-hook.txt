핵심요구사항) useAnalyzeSubmit 훅을 수정하여 Context를 사용할 것.

1) Context import
   - src/features/scan/context/analyze-result-context.tsx에서 useAnalyzeResult 가져오기
   - AnalysisResult 타입 가져오기

2) 기존 로컬 상태 제거
   - const [analysisResult, setAnalysisResult] = useState(...) 제거
   - 해당 import 제거

3) Context 사용
   - const { setAnalysisResult } = useAnalyzeResult() 추가
   - submitAnalyze 함수에서 setAnalysisResult(analysis) 호출은 유지
   - 분석 결과를 Context에 저장하도록 변경

4) 반환 타입 수정
   - UseAnalyzeSubmitReturn 인터페이스에서 analysisResult 제거
   - 다른 반환값들은 유지 (imageData, isLoading, error, setImageData, submitAnalyze, clearError, reset)

5) 타입 정의 정리
   - AnalyzeResponse['analysis'] 타입을 AnalysisResult로 교체
   - 기존 타입과 호환되도록 수정

핵심요구사항) 기존 로직은 최대한 유지하면서 상태 관리만 Context로 변경할 것.
            - submitAnalyze 함수의 로직은 그대로 유지
            - setAnalysisResult 호출 위치 유지
            - 에러 처리 로직 유지
