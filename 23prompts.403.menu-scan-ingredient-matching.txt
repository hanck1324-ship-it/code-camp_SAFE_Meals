조건-파일경로) 구현될 TEST 파일경로: src/components/menu-scan/tests/useIngredientMatching.spec.ts

핵심요구사항) playwright 테스트를 활용하여 TDD 기반으로 구현하고, 테스트에 통과할 때까지 반복할 것.

1) 테스트 제외 라이브러리
   - jest
   - @testing-library/react

2) 테스트 조건
   - timeout은 설정하지 않거나, 500ms 미만으로 설정할 것.
   - 페이지가 완전히 로드된 후 테스트할 것.
       - 페이지 로드 식별 요구사항: 고정식별자 data-testid 대기 방법
       - **중요금지사항** 페이지 로드 식별 금지사항: networkidle 대기 방법

3) 테스트 시나리오
   3-1) 성공 시나리오
        - 레시피 데이터에서 메뉴명으로 정확히 조회되는지 검증
        - 직접 재료가 올바르게 추출되는지 검증
        - 사용자 알레르기와 재료가 정확히 매칭되는지 검증
        - 매칭된 알레르기 성분이 반환되는지 검증

   3-2) 실패 시나리오
        - 존재하지 않는 메뉴명 입력 시 빈 배열 반환 검증
        - 빈 메뉴명 입력 시 에러 처리 검증
        - 레시피 데이터 로드 실패 시 에러 처리 검증

==============================================

핵심요구사항) 직접 재료 매칭 커스텀 훅을 구현할 것.

1) 입력 데이터 구조
   {
     menuName: string;
     userAllergies: string[];
   }

2) 출력 데이터 구조
   {
     directIngredients: string[];
     matchedAllergens: {
       ingredient: string;
       allergen: string;
       reason: string;
     }[];
   }

3) 직접 재료 매칭 로직 상세내용
   3-1) 레시피 데이터 조회
        - 내부 DB/JSON 파일에서 메뉴명으로 레시피 조회
        - 정확한 메뉴명 매칭 (대소문자 무시, 공백 정규화)
        - 레시피가 없는 경우 빈 배열 반환

   3-2) 직접 재료 추출
        - 레시피의 ingredients 배열에서 모든 재료명 추출
        - 재료별 알레르기 성분 정보 추출

   3-3) 알레르기 매칭
        - 각 재료의 allergens 배열과 사용자 알레르기 비교
        - 매칭되는 경우 matchedAllergens에 추가
        - reason 문구 생성: "[메뉴명]에는 [재료명]이 포함되어 있습니다 ([알레르기명] 알레르기)"

4) 레시피 데이터 구조 (src/data/recipes.json)
   [
     {
       menuName: string;
       ingredients: {
         name: string;
         allergens: string[];
       }[];
       category: string;
       cookingMethod: string;
     }
   ]

5) 훅 인터페이스
   const useIngredientMatching = () => {
     const matchIngredients = (input: MatchInput) => MatchOutput;
     return { matchIngredients };
   }

6) 테스트 케이스 예시
   입력:
   {
     menuName: "김치찌개",
     userAllergies: ["갑각류", "해산물"]
   }

   출력:
   {
     directIngredients: ["돼지고기", "김치", "두부", "새우젓", "고춧가루"],
     matchedAllergens: [
       {
         ingredient: "새우젓",
         allergen: "갑각류",
         reason: "김치찌개에는 새우젓이 포함되어 있습니다 (갑각류 알레르기)"
       }
     ]
   }

핵심요구사항) 레시피 데이터 파일을 다음 경로에 생성할 것.
                - 파일경로: src/data/recipes.json
                - 최소 10개 이상의 한식 메뉴 데이터 포함
                - 각 메뉴별 직접 재료 및 알레르기 정보 포함
                - 예시 메뉴: 김치찌개, 된장찌개, 부대찌개, 비빔밥, 불고기, 삼겹살, 김치볶음밥, 냉면, 잡채, 갈비찜
