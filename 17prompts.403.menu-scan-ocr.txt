핵심요구사항) Google Cloud Vision API를 연동하여 OCR 텍스트 추출 커스텀 훅을 구현할 것.

핵심요구사항) 다음 OCR 기능을 step-by-step으로 구현하고, 적용 결과를 체크리스트로 반환할 것.
                1) Google Cloud Vision API 연동
                    - API 키: 환경변수에서 로드 (NEXT_PUBLIC_GOOGLE_VISION_API_KEY)
                    - 이미지 업로드: Base64 인코딩
                    - TEXT_DETECTION 기능 사용

                2) OCR 텍스트 추출
                    - 전체 텍스트 추출 (fullTextAnnotation)
                    - 개별 단어 추출 (textAnnotations)
                    - 언어 감지 (한국어, 영어 우선)

                3) Bounding Box 정보 추출
                    - 각 텍스트의 위치 좌표 (x, y, width, height)
                    - 신뢰도 점수 (confidence)
                    - 텍스트 방향 정보

                4) 에러 처리
                    - API 호출 실패 처리
                    - 네트워크 오류 처리
                    - 재시도 로직 (최대 3회)

핵심요구사항) OCR 결과 타입을 다음과 같이 정의할 것.
                ```typescript
                interface OCRResult {
                  text: string;
                  confidence: number;
                  boundingBox: {
                    x: number;
                    y: number;
                    width: number;
                    height: number;
                  };
                }

                interface OCRResponse {
                  results: OCRResult[];
                  fullText: string;
                  isLoading: boolean;
                  error: string | null;
                }
                ```

핵심요구사항) 훅 인터페이스를 다음과 같이 구현할 것.
                - 입력: Blob (전처리된 이미지)
                - 출력: OCRResponse
                - 메서드: extractText(), retry()
