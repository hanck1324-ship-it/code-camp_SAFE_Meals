핵심요구사항) Context를 생성하여 분석 결과를 전역으로 관리할 것.

1) Context 구조
   - 파일: src/features/scan/context/analyze-result-context.tsx
   - Provider: AnalyzeResultProvider
   - Hook: useAnalyzeResult
   - 상태:
       - analysisResult: AnalysisResult | null
       - setAnalysisResult: (result: AnalysisResult | null) => void
       - clearAnalysisResult: () => void

2) AnalysisResult 타입 정의
   - 파일 내부에 타입 정의
   ```typescript
   export interface MenuAnalysisItem {
     id: string;
     original_name: string;
     translated_name: string;
     description: string;
     safety_status: 'SAFE' | 'CAUTION' | 'DANGER';
     reason: string;
     ingredients: string[];
   }

   export interface AnalysisResult {
     overall_status: 'SAFE' | 'CAUTION' | 'DANGER';
     detected_ingredients: string[];
     warnings: Array<{
       ingredient: string;
       allergen: string;
       severity: 'HIGH' | 'MEDIUM' | 'LOW';
     }>;
     message_ko: string;
     message_en: string;
     results?: MenuAnalysisItem[];
   }
   ```

3) Context 구현 세부사항
   - useState를 사용하여 analysisResult 상태 관리
   - setAnalysisResult 함수: 분석 결과 저장
   - clearAnalysisResult 함수: 분석 결과 초기화 (null로 설정)
   - useAnalyzeResult 훅: Context를 사용하지 않으면 에러 발생

4) Context Provider 적용
   - src/app/(mobile)/layout.tsx에 AnalyzeResultProvider 추가
   - 기존 Provider들을 감싸는 형태로 추가
   - 적용 순서: AuthProvider > AnalyzeResultProvider > children
