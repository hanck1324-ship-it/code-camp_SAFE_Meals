아래의 조건을 모두 적용하여, 아래의 요구사항을 모두 구현할 것.
구현 결과를 체크리스트로 반환할 것.

==============================================

조건-환경) React Native / Expo 환경에서 구현
         - Expo Router 기반 모바일 앱
         - expo-camera 라이브러리 사용
         - React Native 컴포넌트 (View, Text, TouchableOpacity 등) 사용

==============================================

조건-파일경로) 구현될 TSX 파일경로: apps/mobile/app/camera.tsx

==============================================

조건-스타일) React Native StyleSheet를 사용하여 스타일링할 것.
            - StyleSheet.create()로 스타일 정의
            - 색상 일관성 유지:
              * 메인 그린: #22c55e
              * 배경: #ffffff, #000000
              * 텍스트: #1f2937, #6b7280
            - 반응형은 Dimensions API 또는 flex 레이아웃 활용

==============================================

핵심요구사항) 메뉴판 사진 촬영을 위한 카메라 UI를 구현할 것.

1) 카메라 접근 기능
   - expo-camera의 useCameraPermissions 훅 사용
   - 카메라 권한 요청 및 처리
   - 후면 카메라 사용 (facing="back")
   - 권한 거부 시 SafeAreaView로 안내 화면 표시

2) 사진 촬영 기능
   - CameraView.takePictureAsync() 메서드 사용
   - 촬영 버튼 터치 시 사진 캡처
   - 촬영된 이미지를 컴포넌트 내부 상태(useState)로 보관
   - **중요금지사항** 서버로 전송하지 말 것 (다음 단계에서 처리)

3) 미리보기 기능 ⭐ 새로 추가 필요
   - 촬영된 사진을 Image 컴포넌트로 전체 화면 표시
   - 재촬영 버튼 제공 (촬영된 상태 초기화)
   - 사진 삭제 시 다시 카메라 화면으로 전환
   - 미리보기 화면 레이아웃:
     * 상단: 닫기 버튼
     * 중앙: 촬영된 이미지 (전체 화면)
     * 하단: [재촬영] [분석하기] 버튼

4) 분석하기 버튼 ⭐ 새로 추가 필요
   - 사진이 촬영된 상태에서만 활성화
   - 버튼 터치 시 webview로 이동 (결과 페이지)
   - router.push()로 이미지 URI 전달
   - Haptics 피드백 제공

5) 접근성 및 사용성
   - TouchableOpacity로 버튼 구현 (터치 피드백)
   - expo-haptics로 촉각 피드백 제공
   - 카메라 권한 거부 시 안내 메시지 + 권한 요청 버튼
   - SafeAreaView로 노치/상태바 영역 대응

6) UI 구성 요소
   - 카메라 화면:
     * CameraView (전체 화면)
     * 촬영 가이드 프레임 (4개 모서리)
     * 안내 텍스트
     * 촬영 버튼 (하단 중앙, 원형)
     * 닫기 버튼 (우측 상단)

   - 미리보기 화면: ⭐ 새로 구현
     * Image 컴포넌트로 촬영된 사진 표시
     * 닫기 버튼 (우측 상단)
     * 재촬영 버튼 (하단 좌측)
     * 분석하기 버튼 (하단 우측, 강조 스타일)

핵심요구사항) 현재 존재하는 camera.tsx를 리팩토링하여 위 기능을 완성할 것.
            - 기존 CameraView, 권한 처리 로직 유지
            - useState로 촬영된 이미지 상태 추가
            - 미리보기 화면 조건부 렌더링 추가
            - 재촬영 및 분석하기 버튼 구현

핵심요구사항) React Native 스타일로 구현할 것.
            - 전체 화면: flex: 1
            - 버튼 배치: position: 'absolute', bottom 사용
            - 안전 영역: SafeAreaView 사용
            - 터치 피드백: TouchableOpacity 사용
            - 아이콘: @expo/vector-icons의 Ionicons 사용
